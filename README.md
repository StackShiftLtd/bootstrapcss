## Basic Bootstrap Website

A simple responsive website built using **HTML5** and **Bootstrap CSS**.

This project was created as a practice/demo website to explore layout design, responsive components, and modern UI styling using Bootstrap.


## Features

- Responsive design (mobile-friendly)
- Built with Bootstrap 5
- Clean and simple UI layout
- Easy to customize and expand
- 
## Technologies Used

- **HTML5**
- **Bootstrap 5**
- **CSS3**
- (Optional) JavaScript
